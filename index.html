Q&A

Qual framework você ultiliza?

Eu normalmente ultilizo React js, que não é uma framework, é uma biblioteca, mas recentemente venho ultilizando Next js que é considerada uma framework junto de Typescript.

Que tipo de desenvolvedor você é?

Posso trabalhar tanto com Frontend quanto Backend ultilizando de Javascript, então me considero um Fullstack (FE-heavy), já que tenho preferência pelo Frontend.

Ultiliza alguma framework para CSS?

Atualmente venho ultilizando Tailwind por achar mais organizado, contudo aprendi primeiro Bootstrap então posso ultilizar dos dois e opto pelo mais viável de acordo com a situação.

Consegue operar em sistema legado?

Vai depender da tecnologia a qual se refere como legado. Caso seja algo como jQuery consigo operar sem problemas e realizar as mudanças necessárias.
